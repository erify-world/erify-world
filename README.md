## ⚡ Quick Start

### 🚀 For Developers

```bash
# Clone the repository
git clone https://github.com/erify-world/erify-world.git
cd erify-world

# Explore operational templates
cd docs/ops/scheduling/nodejs
npm install
npm start

# Or try Cloudflare Workers
cd ../cloudflare-workers
npm install
npm run dev
```
